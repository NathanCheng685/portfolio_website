VStandby Studio

A real-time AI digital human that watches livestreams alongside you · BETA AI Hackathon · Apr 2026

VStandby Studio is a companion AI for livestream viewers. It detects high-motion highlights with computer vision, threads recent caption context to an LLM, and delivers spoken commentary — with selectable persona modes so different channels feel distinct.

  • WebSocket pipeline streams highlights to the LLM with configurable playback delay (deliverAfterMs)
  • OpenCV motion detection (highlight_detect.py) identifies in-stream peak moments
  • Persona-aware prompting renders distinct speaking styles per channel
Python OpenCV WebSockets LLM API Real-time Systems
Welldify AI Wellbeing Coach

Diary-based LLM for personalized mental wellness · contributed during internship, 2025

Welldify is a Hong Kong–based AI wellness platform — a diary-driven LLM coach delivering personalized recommendations grounded in evidence-based positive psychology (PERMA, DASS). Users journal regularly; the model surfaces patterns and prompts trackable progress on individual wellness goals.

I contributed on the model side — supporting LLM training monitoring across 66,000 labeled sentences for wellbeing-index calibration, and helping refine the framework that backs how the system reads emotional state.
